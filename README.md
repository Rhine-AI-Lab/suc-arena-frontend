
# Suc Arena Frontend

<br/>

## Environment

在 `https://nodejs.org/en` 安装 NodeJs。

推荐使用 `20.x.x` LTS版本。


```text
# clone 仓库
git clone https://github.com/Rhine-AI-Lab/suc-arena-frontend

# 进入目录
cd suc-arena-frontend

# 安装依赖
npm i
```

## Quick Start


```text
npm run dev
```

浏览器访问 `http://localhost:3310/show` 即可


## SSR Deploy


```text
npm run build
npm run start
```

浏览器访问 `http://localhost:3310/show` 即可
